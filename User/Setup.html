Setup Practice Management Software
Quickly implement practice management software in your CA, CS, CPA Practice
1. Invite Staff

This video would help you to manage staff on easycloudbooks.

2. Import Services from Market module

This video would help you to create service on easycloudbooks.
Or Import Services from Market. There are many default Services available in Market Place.

3. Import Companies from excel.

This video would help you to import companies on easycloudbooks. You should sort your company list before importing it on easycloudbooks sothat duplication can be avoided.

4. Create Team with default Manager

This video would help you to create team on easycloudbooks.

5. Set Year and Period, which will help you to create and manage projects.

This video would help you to set Year and Period on easycloudbooks.
After half an hour of Company Activation, Year & Periods automatically created.

6. Create Multiple Projects.

This video would help you to create Multiple Projects on easycloudbooks. You can create 500 projects at a time.
Projects combination should be -
1. 50 companies + 1 service + 1 year.
2. 25 companies + 1 service + 2 year.
3. 25 companies + 2 service + 1 year.

Note :- Set Price before create Projects.
