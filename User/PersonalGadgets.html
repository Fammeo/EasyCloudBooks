Personal Gadgets

Personal Gadgets is used to track important metrics for quick reference and control purpose. You can create gadgets by creating queries through queries builder module and then applying that query in creating gadget.

How to Create 'Cash & Bank Balance' Personal Gadget ?

Step 1: For achieve this at first you need to create related query by following below steps
- Goto query builder page
- Click on create new button on query builder page.
- Type proper name of query like Cash and Bank Balance so that we can use it for filter in future.
- In builder section choose parameter “Account : type equal Bank Accounts”,
click +add rule and choose “Account : type equal Cash in hands”,
click +add rule and choose “Account : type equal Bank OD Accounts”, select OR operator between them.
- You can create personal or shared type query.
- Save this query for future use.

Step 2: - Now goto personal gadgets page and click on + Gadgets button for creating new gadget.
- Write proper name of gadget like Cash and Bank Balance.
- Create new group or select existing group (if you create one before) from drop down.
- Click on +button for adding scheduler or type initials for select saved scheduler from drop down.
- Select type of objects from the dropdown in parameter section. In our case we select ‘Account’ .
- Select type of sub object from the dropdown in parameter section. In our case we select ‘Total Balance’ .
- Select type of value from the dropdown. In our case we select ‘Total’ .
- Select query builder search in builder section, then select equal then type initials for selecting saved query from the dropdown. In our case we select Cash and Bank Balance query.
- On Is reverse switch if this gadget's value getting low means good.
- Write unit of result.
- Finally click on save changes button.

- Now you can see Cash and Bank Balance gadget on screen.
- There are many buttons on this gadgets, like view results,copy gadgets,edit gadgets, refresh now and delete gadget.
- You can change group name by click on edit group button.
- Click on view result button for view final result of query.
- It is redirected to pages according to query.
- All Gadgets which are created previously shown here on personal gadgets page.

How to find 'Project Status or Project Stages' in Personal Gadgets ?

Step 1: For achieve this at first you need to create related query by following below steps
- Goto Query Builder page and click on create new button.
- Type proper name of query like “In Progress Work” in our case so that we can use it for filter in future.
- In builder section choose parameter Project : Status equal to In Progress. (There are 7 options in dropdown, which option you want to find select, in our case we select ‘In Progress’ )
- AND operator automatically selected in single query parameter.
- You can create personal or shared type query.
- Save this query for future use.

Step 2: - Now goto personal gadgets page and click on + Gadgets button for creating new gadget.
- Write proper name of gadget like ‘Projects In Progress’.
- Create new group or select existing group (if you create one before) from drop down.
- Click on +button for adding scheduler or type initials for select saved scheduler from drop down.
- Select type of objects from the dropdown in parameter section. In our case we select ‘Project’.
- Select type of sub object from the dropdown in parameter section. In our case we select ‘Project Count’.
- Select query builder search in builder section, then select equal then type initials for selecting saved query from the dropdown. In our case we select ‘In Progress Work’.
- On ‘Is reverse’ switch if this gadget's value getting low means good.
- Write unit of result.
- Finally click on save changes button.

- Now you can see ‘Projects In Progress’ gadget on screen.
- There are many buttons on this gadgets, like view results,copy gadgets,edit gadgets, refresh now and delete gadget.
- You can change group name by click on edit group button.
