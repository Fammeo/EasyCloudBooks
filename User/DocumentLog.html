DocumentLog

This module tracks every movement of documents and folders. You will be able to see who uploaded, deleted, moved etc. documents and folders. Further you can use filter and custom filter as well to filter the logs and you can also create personalised gadgets to track number of documents upload or number of documents deleted etc. Further if you want to recover documents deleted then this feature is also available.

Which actions we can track in document log section ?

User can track all actions such as upload files, delete files, delete folder, duplicate folder, duplicate file, download file, create folder, create file, view file, edit file, move file, copy file, move folder, copy folder, rename folder, rename file, recover file, recover folder etc in document log section.

If any file or folder delete by mistake, can i recover it ?

yes , user can recover any file or folder which are deleted, just go to document log section, find your file or folder, preview it and click on recover button. Deleted files or folder restore in your document section in folder named recovered folder. User can recover deleted files and folders within 30 days of deletion. After this period you can not recover any file or folder.

Till how many days we can recover deleted files ?

User can recover deleted files and folders within 30 days of deletion.

Can user directly download files from document log section ?

Yes, user can preview desired file and download it directly from document log section.

Can user filter by company or user name or filename or action on files in document log section ?

Yes, user can use custom filter which contains many parameter which make search easy. User can directly search by company name, user name, file name, and action on files in document log section.
