Project Templates

On project template page you will find different todos, which can later be used in future for making another projects. you will see several number of todos listed on viewing the template from inside. Todos are the list of notes that are written to follow an structured development of your project. It can be used further in your new project as well.

How to create a Project Template ?

step 1: On the template page click on TEMPLATES button. you will be redirected to next page where you can create new template by entering template name and selecting services.

step 2: After clicking on TEMPLATES button, here you will find fields where you have to enter template name and select appropriate service. After filling the fields, click CREATE TEMPLATE button, which will create a new template for you.

How to view Project Templates and manage Todos ?

As seen in picture you can view your template by clicking BLUE view button. You will be redirected inside the template page, where you will see todos and notes which are explained as follows.

After clicking on the view button, here you will find your selected template's todos and notes if any. Todos are list of notes which are to be followed in certain way to build or followup your projects. These todos can also be used in future projects if you want your project to follow this same todos. You can take several action on todos like PRIORITIZING, DELETING, EDITING and SET APPLICABLE which will move it to down the field.

On the same page scrolling down, you will find template's notes if any. Notes are any messages that one wants others to know who are working on the same project. Only authorized can write notes.

How search filter works ?

step 1: On the template page click on SHOW FILTER button. you will be redirected to next page where you find service and title input boxes.

step 2: After clicking on SHOW FILTER button, here you will find fields where you have to enter service(only 3 letters to automatically search for you) and template title(only 3 letters to automatically search for you), and suggestion for you will be automatically dropped down for you.

How to delete Project Template ?

To delete the current template user has been provided with RED button as seen in picture.

What is meaning of related on advance button of Project template access ?(In Role Page)

When advance button is OFF on template, user will be able to templated for only for services belonging to his team. When advance button is ON, user will be able to create templates for all the services.
