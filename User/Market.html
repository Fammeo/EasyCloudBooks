Market

If you find it hectic to create each service manually, then don’t worry we have Market Place from there you can import Services.
We have already created few Packages which consists Services, Schedulers, Automations, Custom Fields, Queries, Gadgets and Templates.
Let’s see how we can import a package from market.
1. Go to market page, choose one package which you want to import to your company.
2. Click on Get details Button.
3. It will open one popup which will display the details about that package.
4. Clicking on “Import to my Company” button will add all the items present in package to your company.
5. Once package gets imported fully then you can see imported Service.
