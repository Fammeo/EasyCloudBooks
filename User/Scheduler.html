Scheduler

Scheduler is more like project scheduler. Scheduler will help to create trigger events based on recurrence date. Presently scheduler will be used for setting frequency by which gadgets get updated on dashboard for eg. weekly, monthly, daily etc. In future it will be used in email sending, sms sending etc.

Can I start scheduler from future or past date?

Scheduler can be started from today or future date but it can not be started from past dates.
For eg. If today is 30th Jan 2019 and you want to start monthly recurring projects starting from 1st May 2019 you can do that. In short scheduler can start from today onwards.

Can I set recurrence bi-yearly or bi-monthly or bi-weekly etc ?

You can set recurrence as per your choice. For instance to set -
Bi-yearly recurrence - Set repeat yearly and set every 2 years.
Bi-monthly recurrence- Set repeat monthly and set every 2 months.
Quarterly recurrence- Set repeat monthly and set every 3 months.
Bi- Weekly recurrence- Set repeat weekly and set every 2 weeks and select the day.
Every alternate day- Set repeat daily and set every 2 days.

Can I set recurrence for Monday and Tuesday every week ?

You can set recurrence to create projects twice a week with specific days. For this purpose set repeat weekly and set every 1 week and then select monday and tuesday. It will set recurrence for every monday and tuesday. If you want to set recurrency for monday and tuesday every two week. Then set repeat weekly and set every 2 week and then select monday and tuesday.

Can I set recurrence yearly, monthly with specific date or specific day ?

You can set recurrence yearly or monthly with specific date or day. Yearly recurrence- set repeat yearly and set every 1 year and then select on specific date or specific day of month. For instance you can set recurrence like repeat project every year on 1st monday of february like this image.

How long I can set recurrence of projects ?

You can set end of recurrence in different ways.
For instance -
You can set recurrence never end - Click End Never.
You can set recurrence end after particular number of occurrence - Click End After and then select number of occurrence.
You can set recurrence end on particular Date - Click End On Date and then select particular date when recurrence should stop.

Explain Access control on Scheduler page ? (on Role Page)

Super owner can control which user can see scheduler page and what he can do on that pages. If switch is On then user can open, create, edit & delete schedulers.
