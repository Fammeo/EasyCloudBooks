Query

We are introducing new page for managing queries, whereby user will be able to reply on open queries for him. Further query manager (person who is given this right on role page) will be able to view his queries as well as queries for other users to make him able to resolve queries faster.

Can i get notification on Query every time when anyone Comment a Query message ?

Yes, you get notification if you enable Notify Me button under User Profile edit setting page.

Can i select to Turn on or Turn off notification on particular Query ?

Yes, you can turn on or turn off notification on particular query. For this purpose you need to go inside that particular query and turn on or turn off notification as per your requirement.

What access I can control on Query Page ?(In Role Page)

Through role on query page, you can control who can see query page and what he can do there.

If QUERY page is ON, user will be able to go to Query page and he will be able to view queries related to him i.e. queries asked to him and queries asked by him.
If CREATE button is ON, user will be able to raise queries on query page. However user has default right of raising query on projects on which he is project member.
If ADVANCE button is ON, user will be able to view queries of others users as well. Further user will be able to Close the queries of other users if no discussion occurred for last 30 days.

What is use of Advance role for Query Page ?(In Role Page)

Queries need to be solved fast and if you want to assign someone with duty of making sure about it, you can give him advance role. With this role user will be able to view queries of all the person and manage the queries. This will reduce the burden of overall management of practice.

What is meaning of related on advance button of Query page access?(In Role page)

When advance button is OFF, related queries mean user will be able to view only those queries which are asked to him or asked by him. When advance button is ON, user will be able to view queries related to other users as well.

Can I Bookmark queries to for quick reference?

Yes you can bookmark queries and filter the queries only to show bookmarked queries. This functionality you can use to keep reference of queries you are currently working on or to keep track of important queries for faster processing or to keep track of queries requiring discussion.
