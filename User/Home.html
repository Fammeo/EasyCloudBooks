Dashboard

Dashboard is the information page that a user can see just after his/her login. This page contains various information regarding user's project progress as per the roles he is associated with the project. Dashboard will vary its information as per user. There are total 10 informational gadgets shown on this page which are Project Deviation, Due Date, Project Completed, Top 10 Project Tags, Top 10 Account Tags, Top 10 Company Tags, Top 10 Contact Tags, Pending Project By Project Managers, Project Status Today, Accounts Status Today.

What is a Dashboard ?

A Dashboard is a user interface that organizes and presents information in a way that is easy to understand.
